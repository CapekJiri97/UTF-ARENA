UTF Arena — MVP

Run:
- Open `index.html` in your browser (double-click or use a local server).

What this initial MVP includes:
- Top-down 2D canvas with black background and UTF-like glyphs.
- WASD movement, mouse aiming, click to shoot projectiles.
- Basic towers placed in a circle.
- Minions spawn from towers and move toward enemy towers.
- Simple HUD with HP / Gold / EXP and a minimap.
